Difference between HTML and HTML5

HTML HTML5
Based on a hierarchical structure using div and span elements for layout. Introduces new structural elements like <header>, <nav>, <section>, and <footer> for improved document structure.
Limited support for multimedia via plugins like Adobe Flash. Native support for audio and video elements, eliminating the need for external plugins.
Limited form input types and attributes. Expanded form input types (e.g., email, URL, range) and enhanced form validation with attributes like required and pattern.
